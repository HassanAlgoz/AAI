## Dos

These guidelines are based on standard taxonomy for multiple-choice item writing as defined by Haladyna, Downing, and Rodriguez (2002) in their consensus of validity evidence for item-writing rules.

**Generate multiple-choice questions adhering strictly to the following construction rules:**

1. **Use Unique Vocabulary:** The correct answer should be paraphrased rather than repeating exact words from the stem. Alternatively, if a technical term must be used, include it in every option to eliminate keyword association clues.
2. **Maintain Parallel Plausibility:** Ensure all options (distractors and the correct answer) share the same level of grammatical complexity, tone, and use of qualifiers. If the correct answer requires "hedging" (e.g., *usually*, *may*), ensure distractors utilize similar language to avoid standing out.
3. **Ensure Grammatical Consistency:** Write the stem so that it creates a complete, grammatically correct sentence with **every** option provided. Check specifically for article agreement (a/an) and singular/plural consistency across all choices.
4. **Standardize Option Length:** Keep all options approximately the same word count and level of detail. If the correct answer requires specific technical precision, ensure distractors are written with equal precision and length.
5. **Create Distinct Options:** Ensure every option represents a distinct, independent concept. Avoid "convergence" where the correct answer is simply a combination of elements found in the distractors.
6. **Order Values Logically:** When using numerical options, arrange them in a consistent order (ascending or descending). Ensure the correct answer is randomly distributed among the high, low, and middle values rather than consistently clustering in the center.
7. **Homogenize Answer Choices:** Ensure all options belong to the same category or logical grouping. If utilizing opposites, include multiple pairs or ensure the distractors are equally distinct to prevent the "odd one out" or "opposite pair" heuristic.
8. Use **neutral wording**; no trick questions.

## Dont's

Avoid the common pitfalls that inadvertently reveal the correct answer:

1. Absolutes vs. Qualifiers: Correct answers often use "hedging" terms like usually, often, or may. Conversely, distractors that use "absolute" terms like always, never, or all are frequently incorrect because they are easier to disprove.

2. Grammatical Cues: The lead-in sentence (the "stem") must grammatically match the options. If the stem ends in "an," but only one option starts with a vowel, that option is likely the answer.

3. Length and Detail: The "Longest Answer" rule is common. Item writers often include more detail, qualifications, and precision in the correct choice to ensure it is indisputably right, making it stand out from shorter, vaguer distractors.

4. Word Repeats: If a specific technical word or phrase appears in both the stem and only one of the options, it creates a "clang association" that guides the test-taker to the correct choice.

5. The "Middle" Value: In numerical questions, the correct answer is rarely the highest or lowest value in the set; it is frequently found in the middle of the range provided.

6. Convergence Strategy: If multiple options share similar elements (e.g., A: Red Square, B: Red Circle, C: Blue Square), the answer is likely the one that combines the most frequent elements (Red Square), as distractors are often built by slightly modifying the correct answer.

7. Mutually Exclusive Pairs: If two options are direct opposites, one of them is statistically more likely to be the correct answer.
