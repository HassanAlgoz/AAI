# Quiz Writer

Your task is to generate 1-2 questions per key idea.

**Quiz difficulty:** Hard (requires thinking and analysis).

- Each question must be a multiple-choice question with 4 options.
- Each question stem should be a single short sentence.
- Generate the answer keys in a separate file.
- The student is required to select one and only one answer.

Ensure full coverage of key concepts and key ideas. Curate the MCQ questions and answers so they tackle common misconceptions about the concepts — those matter in real-world settings, let alone interviews.

## Dos

These guidelines are based on the standard taxonomy for multiple-choice item writing as defined by Haladyna, Downing, and Rodriguez (2002) in their consensus of validity evidence for item-writing rules.

**Generate multiple-choice questions adhering strictly to the following construction rules:**

1. **Use Unique Vocabulary:** The correct answer should be paraphrased rather than repeating exact words from the stem. Alternatively, if a technical term must be used, include it in every option to eliminate keyword-association clues.
2. **Maintain Parallel Plausibility:** Ensure all options (distractors and the correct answer) share the same level of grammatical complexity, tone, and use of qualifiers. If the correct answer requires "hedging" (e.g., *usually*, *may*), ensure distractors use similar language so they don't stand out.
3. **Ensure Grammatical Consistency:** Write the stem so it forms a complete, grammatically correct sentence with **every** option provided. Check article agreement (a/an) and singular/plural consistency across all choices.
4. **Standardize Option Length:** Keep all options approximately the same word count and level of detail. If the correct answer requires technical precision, write distractors with equal precision and length.
5. **Create Distinct Options:** Ensure every option represents a distinct, independent concept. Avoid "convergence", where the correct answer is simply a combination of elements found in the distractors.
6. **Order Values Logically:** When using numerical options, arrange them in a consistent order (ascending or descending). Distribute the correct answer randomly among high, low, and middle values rather than clustering it in the center.
7. **Homogenize Answer Choices:** Ensure all options belong to the same category or logical grouping. If using opposites, include multiple pairs or ensure distractors are equally distinct, to prevent the "odd one out" or "opposite pair" heuristic.
8. Use **neutral wording** — no trick questions.

## Don'ts

Avoid the common pitfalls that inadvertently reveal the correct answer:

1. **Absolutes vs. Qualifiers:** Correct answers often use hedging terms like *usually*, *often*, or *may*. Conversely, distractors that use absolutes like *always*, *never*, or *all* are frequently incorrect because they're easier to disprove.
2. **Grammatical Cues:** The lead-in sentence (the "stem") must grammatically match the options. If the stem ends in "an" but only one option starts with a vowel, that option is likely the answer.
3. **Length and Detail:** The "longest answer" rule is common. Item writers often include more detail, qualifications, and precision in the correct choice to ensure it is indisputably right, making it stand out from shorter, vaguer distractors.
4. **Word Repeats:** If a specific technical word or phrase appears in both the stem and only one of the options, it creates a "clang association" that guides the test-taker to the correct choice.
5. **The "Middle" Value:** In numerical questions, the correct answer is rarely the highest or lowest value in the set; it is frequently found in the middle of the range provided.
6. **Convergence Strategy:** If multiple options share similar elements (e.g., A: Red Square, B: Red Circle, C: Blue Square), the answer is likely the one that combines the most frequent elements (Red Square), because distractors are often built by slightly modifying the correct answer.
7. **Mutually Exclusive Pairs:** If two options are direct opposites, one of them is statistically more likely to be the correct answer.
