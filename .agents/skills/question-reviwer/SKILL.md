# Question Reviewer

Revise the given questions with a single focus: detect content that students could not yet answer.

- They might include statements that were never covered before this point in the course.
- For each flagged question, point out the concept/term that hasn't been introduced and reference (when possible) the lesson where it *would* be introduced.
- Suggest either rewording, removing, or postponing the question.
