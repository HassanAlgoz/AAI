# Lesson Hooks

Five ways to start a lesson with a hook:

1. Start with a really interesting question.
2. Use a surprising statement:
   1. a statistic,
   2. a fact that very few know about, or
   3. a bold claim to be verified.
3. A relatable story.
4. A relatable meme (with alt text).

Use one of the above hooks in the introduction, depending on the context of the lesson itself. If no hook fits, just stick to listing the *learning objectives* right away.
