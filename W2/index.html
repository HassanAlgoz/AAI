    Module 1: Introductions

    Learning outcomes:

    1. Define Data Science
    2. Identify the three circles that make up Data Science
    3. Recognize the three roles that a Data Scienist takes on
    4. Distinguish between Data Engineers, Data Analysts and Machine Learning Engineers
    5. Define Data Analysis
    6. Distinguish between CDA and EDA and give one example for each

    Lessons and labs:

    1. Introduction to Data Science Processes and Roles
    2. Lab 1: Confirmatory Data Analysis (CDA)- blind typing experiment
    3. Lab 2: Exploratory Data Analysis (EDA)- US Census 2010-2019

    Exercises:

    1. John Snow (3 questions)

    Module 2: Data Wrangling

    Learning outcomes:

    1. Explain the roles of DataFrame and Series in pandas and inspect basic metadata (head, tail, info, dtypes, index, columns)
    2. Load tabular data into pandas from local files and URLs, and construct a DataFrame from Python data structures
    3. Select and filter data using column selection, boolean masks, combined conditions, and label/position indexing with loc and iloc
    4. Create and transform columns using vectorized operations, including deriving new columns, dropping columns, and renaming columns
    5. Compute descriptive statistics with methods such as mean, median, describe, and agg, while interpreting grouped summaries
    6. Aggregate data by one or more categories using groupby and count category frequencies with value_counts
    7. Apply custom functions table-wise, row/column-wise, and element-wise using pipe, apply, and map

    Lessons and labs:

    1. Lab: Pandas Essentials
    2. Lab: Pandas Summary Statistics
    3. Lab: Pandas Function application
    4. Lab: Bike Sharing

    Exercises:

    1. Getting to Know your Data:
      1. World Food Facts
      2. Chipotle
      3. Occupation
    2. Filtering and Sorting Data
      1. Chipotle
      2. Euro12
      3. Fictional Army
    3. Group By
      1. Alcohol Consumption
      2. Occupation
      3. Regiment

    Module 3: Descriptive Statistics

    M3-1: Statistical Analysis

    Learning outcomes:

    1. Define statistics
    2. Distinguish between Population and Sample
    3. Give examples of numerical variables and categorical variables
    4. Recognize the two type of summaries: numerical and visual
    5. Explain what Proportion is
    6. Explain what a Distribution is
    7. Describe what are the mean and variance
    8. Explain what a Percentile means
    9. Describe how do we get a z-score and why we might need it
    10. Explain what is 1, 2, and 3 standard deviations mean for a normal distribution
    11. Given a box-and-whiskers plot, identify its graphical elements and tell what they mean
    12. Given a plot like the Comparative Sex IQ Distribution, make at least 2 interpretations
    13. Given a plot like the Comparative Sex IQ Distribution, recognize whether a statement is true or not
    14. Name at least 2 outlier-removal methods and explain how they work
    15. Given a distribution plot; name it (e.g., normal, uniform, or skewed to the right)

    Lessons:

    • Lesson: Statistics
    • Lab: Statistics

    M3-2: Statistical Visualization

    Learning outcomes:

    1. Distinguish between figure and axes in matplotlib
    2. Name the three figure-level functions in seaborn
    3. Distinguish between Histogram and Barplot
    4. Given a DataFrame, make subplots via the col and row parameters in a figure-level function
    5. Given a DataFrame, make subplots and put custom plots using axes-level functions; and have them do sharex=True or sharey=True
    6. Given a DataFrame, plot the ECDF and interpret it
    7. Given a DataFrame, plot a bi-variate histogram and interpret it
    8. Visualize numerical vs categorical relationships using sns.catplot()
    9. Visualize multiple categorical variables in one plot
    10. Use different colors, markers, and styles in a plot
    11. Relate variables using scatter plot
    12. Identify issues in a given visualization and suggest improvements

    Lessons and labs:

    1. Python Visualization Tools
    2. Visualizing distributions of data
    3. Visualizing categorical data
    4. Visualizing statistical relationships
    5. Effective communication of exploratory results

    Exercises:

    1. EDA Penguins (7 questions)
    2. Visualization Exercises (4 questions)

    Module 4: Inferential Statistics

    M4-1: Hypothesis Testing

    Learning outcomes:

    1. Distinguish between Inferential and Descriptive Statistics
    2. Distinguish between Null and Alternative Hypothesis
    3. Given an alternative hypothesis select the correct statistical test
    4. Intrepret the result of a statistical test (p-value)
    5. Understand that alpha (the significance level) is selected by convention to have the value of 0.05
    6. Distinguish between Treatment and Control Groups
    7. Given a claim in layman terms, describe it in statistical terms, as a hypothesis
    8. Given a hypothesis in statistical terms, explain it in layman terms
    9. Recognize when the test is two-sided or one-sided (and in which direction)

    Lessons:

    • Lesson: Hypothesis Testing
    • Lab: Hypothesis Testing

    M4-2: Association and Correlation

    Learning outcomes:

    1. Distinguish between association and correlation
    2. Recognize how Chi-square test and Cramer’s value are used together
    3. Interpret a Chi-square statistic
    4. Interpret a Cramér’s V statistic
    5. Find the association between two categorical variables in terms of both: significance and strength
    6. Interpret an r-square statistic
    7. Given a scatter plot, choose the corresponding r-square value from a list of given values
    8. Recognize non-linearity and outlier issues with r-squared value
    9. Explain the phrase “correlation does not imply causation” and give at least 1 example of your own
    10. Explain what a Confounding Variable is

    Lessons:

    • Lesson: Correlation
    • Lab 1: Categorical Association
    • Lab 2: Numerical Correlation

    1. Hypothesis Testing (4 questions)
