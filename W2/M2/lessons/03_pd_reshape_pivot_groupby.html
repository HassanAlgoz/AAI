Data Formats

A dataset can be written in two different formats: wide and long.

  • A wide format contains values that do not repeat in the first column.
  • A long format contains values that do repeat in the first column.

For example, consider the following two datasets that contain the exact same data expressed in different formats:

Both datasets contain the exact same information about the teams, but they’re simply expressed in different formats.

The pivot() function is purely reshaping of the data: a single value for each index/column combination is required.

We can use the following basic syntax to convert a pandas DataFrame from a long format to a wide format:

pd.pivot(df, index='player', columns='team', values='points')

Air Quality Data

The air-quality data is provided in a so-called long format data representation with each observation on a separate row and each variable a separate column of the data table. The long/narrow format is also known as the tidy data format.

The air-quality data set has the following columns:

  • city: city where the sensor is used, either Paris, Antwerp or London
  • country: country where the sensor is used, either FR, BE or GB
  • location: the id of the sensor, either FR04014, BETR801 or London Westminster
  • parameter: the parameter measured by the sensor, either 𝑁⁢𝑂2 or Particulate matter
  • value: the measured value
  • unit: the unit of the measured parameter, in this case ‘µg/m³’

and the index of the DataFrame is datetime, the datetime of the measurement.

Let’s say we want the mean concentrations for NO2 and PM2.5 in each of the stations in table form.

In the case of pivot(), the data is only rearranged. When multiple values need to be aggregated (in this specific case, the values on different time steps), pivot_table() can be used, providing an aggregation function (e.g. mean) on how to combine these values.

air_quality.pivot_table(
    values="value",
    index="location",
    columns="parameter",
    aggfunc="mean",
)

def group_mean(df, group_col, value_col, rename_col=None, aggfunc="mean"):
    result = df.groupby(group_col)[[value_col]].agg(aggfunc)
    if rename_col is not None:
        result.rename(columns={value_col: rename_col}, inplace=True)
    return result

no2_group = group_mean(air_quality[air_quality["parameter"] == "no2"], "location", "value", rename_col="no2")
pm25_group = group_mean(air_quality[air_quality["parameter"] == "pm25"], "location", "value", rename_col="pm25")
aq_pivot = no2_group.join(pm25_group, how="outer", lsuffix="_no2", rsuffix="_pm25")

Let’s say now we want to collect all air quality NO2 measurements in a single column (long format). The parameters passed to pandas.melt() can be defined in more detail:

no_2 = no2_pivoted.melt(
    id_vars="date.utc",
    var_name="id_location",
    value_vars=["BETR801", "FR04014", "London Westminster"],
    value_name="NO_2",
).set_index("date.utc")

The additional parameters have the following effects:

  • value_vars defines which columns to melt together
  • value_name provides a custom column name for the values column instead of the default column name value
  • var_name provides a custom column name for the column collecting the column header names. Otherwise it takes the index name or a default variable

Hence, the arguments value_name and var_name are just user-defined names for the two generated columns. The columns to melt are defined by id_vars and value_vars.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from matplotlib.widgets import Button

# enable interactive backend
%matplotlib widget

# start from the existing df (from previous cell) and make coordinates float
editable_df = df.copy().reset_index(drop=True).astype({"x": "float64", "y": "float64"})

fig, ax = plt.subplots(figsize=(6.5, 4.2))
points = ax.scatter(editable_df["x"], editable_df["y"], s=60, color="#1f77b4", picker=True)
reg_line, = ax.plot([], [], color="#d62728", linewidth=2)
text = ax.text(0.02, 0.95, "", transform=ax.transAxes, va="top")
ax.set_title("Drag or double-click to add points")
ax.grid(True, alpha=0.3)

selected_idx = None
press_event = None


def update_limits():
    xr = editable_df["x"].to_numpy()
    yr = editable_df["y"].to_numpy()
    if xr.size == 0:
        return
    xrange = xr.max() - xr.min()
    yrange = yr.max() - yr.min()
    xpad = max(xrange * 0.2, 1)
    ypad = max(yrange * 0.2, 1)
    ax.set_xlim(xr.min() - xpad, xr.max() + xpad)
    ax.set_ylim(yr.min() - ypad, yr.max() + ypad)


def update_reg():
    if len(editable_df) < 2:
        reg_line.set_data([], [])
        text.set_text("")
        fig.canvas.draw_idle()
        return
    m, b = np.polyfit(editable_df["x"], editable_df["y"], 1)
    xmin, xmax = editable_df["x"].min(), editable_df["x"].max()
    reg_line.set_data([xmin, xmax], [m * xmin + b, m * xmax + b])
    preds = m * editable_df["x"] + b
    mae = np.mean(np.abs(editable_df["y"] - preds))
    ss_res = np.sum((editable_df["y"] - preds) ** 2)
    ss_tot = np.sum((editable_df["y"] - editable_df["y"].mean()) ** 2)
    r2 = 1 - ss_res / ss_tot if ss_tot != 0 else float("nan")
    text.set_text(f"y = {m:.3f}x + {b:.3f}\nMAE = {mae:.3f} | R² = {r2:.3f}")
    fig.canvas.draw_idle()


update_limits()
update_reg()


def on_pick(event):
    global selected_idx, press_event
    if event.artist != points:
        return
    selected_idx = event.ind[0]
    press_event = event.mouseevent


def on_motion(event):
    global selected_idx, press_event
    if selected_idx is None or press_event is None:
        return
    if event.inaxes != ax or event.xdata is None or event.ydata is None:
        return
    editable_df.loc[selected_idx, "x"] = float(event.xdata)
    editable_df.loc[selected_idx, "y"] = float(event.ydata)
    points.set_offsets(editable_df[["x", "y"]].to_numpy())
    update_limits()
    update_reg()


def on_release(event):
    global selected_idx, press_event
    selected_idx = None
    press_event = None


def on_double_click(event):
    if not event.dblclick or event.button != 1:
        return
    if event.inaxes != ax or event.xdata is None or event.ydata is None:
        return
    new_idx = len(editable_df)
    editable_df.loc[new_idx, "x"] = float(event.xdata)
    editable_df.loc[new_idx, "y"] = float(event.ydata)
    points.set_offsets(editable_df[["x", "y"]].to_numpy())
    update_limits()
    update_reg()


fig.canvas.mpl_connect("pick_event", on_pick)
fig.canvas.mpl_connect("motion_notify_event", on_motion)
fig.canvas.mpl_connect("button_release_event", on_release)
fig.canvas.mpl_connect("button_press_event", on_double_click)

plt.show()
