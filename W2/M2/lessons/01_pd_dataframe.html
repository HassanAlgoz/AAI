Pandas Essentials

Source:

Pandas provides fast, flexible, and expressive data structures designed to make working with “relational” or “labeled” data both easy and intuitive. Pandas is also very fast since it builds on top of NumPy, which builds on top of C.

The community agreed alias for pandas is pd, so it has been standard practice:

import pandas as pd

Read data in

Data can be loaded into a Pandas DataFrame from various sources and different formats:

  1. Databases: relational (SQL) and unstructured (NoSQL)
  2. Surveys: Google Forms, SurveyMonkey
  3. APIs: Social Media Platforms, Open Street Maps, Flights, ..etc.
  4. Web Scraping: BeautifulSoup, Scrapy, Selenium
  5. Raw Files: csv, excel, txt, json, xml, pdf, ..etc.

Pandas provides the read_csv() function to read data stored as a csv file into a pandas DataFrame. pandas supports many different file formats or data sources out of the box (csv, excel, sql, json, parquet, …), each of them with the prefix read_*.

See: IO tools (text, CSV, HDF5, …) for a table containing available readers and writers.

help(pd.read_csv)

The result of a read_* operation is a DataFrame (usually abbreviated as df), which is essentially a table of rows and columns.

Let’s demonstrate:

read from file on disk

# Best practice: use pathlib to handle file paths
from pathlib import Path

# Set the path to the directory containing the datasets
data_dir = Path('../../data')

# Debugging
print(data_dir.resolve())
print(data_dir.exists())
data_wpm = data_dir / 'wpm.csv'

# Debugging
print(data_wpm.exists())
print(data_wpm.resolve())

Once we have specified the path, we load the data into a Python object; i.e., a DataFrame:

# Notice we use `df` to denote a DataFrame
df_1 = pd.read_csv(data_wpm)

# Debugging

Creating a DataFrame

Sometimes, like in demonstration, you need to create a DataFrame object, not loaded from outside.

DataFrames can be constructed from Python dictionaries, like so:

df = pd.DataFrame(
    {
        'A': [10, 20, 30],
        'B': [40, 50, 60]
    }
)

df
A B
0 10 40
1 20 50
2 30 60
  • Keys 'A' and 'B' become column labels
  • Values become cell values

combine row selection conditions

When combining multiple conditional statements, be careful:

  1. each condition must be surrounded by parentheses ()
  2. you can not use or/and but need to use the or operator | and the and operator &.
# Condition 1
mask_1 = df_titanic["Age"] > 35

# Condition 2
mask_2 = df_titanic["Pclass"].isin([2, 3])

# Combine conditions
mask = mask_1 & mask_2

# Apply the mask to the DataFrame
above_35_df = df_titanic[mask]

select specific rows AND columns

Let’s say, we’re interested in the names of the passengers older than 65 years.

When selecting both rows AND columns, We can’t just use the brackets [] notation directly, rather, we need to preceed add a loc or iloc method:

  1. Use loc for label-based selection (using row/column names)
  2. Use iloc for position-based selection (using table positions)

loc: label-based selection

seniors_names = df_titanic.loc[df_titanic["Age"] > 65, "Name"]
seniors_names.head()

96                Goldschmidt, Mr. George B
116                    Connors, Mr. Patrick
493                 Artagaveytia, Mr. Ramon
630    Barkworth, Mr. Algernon Henry Wilson
Name: Name, dtype: str

When using either loc or iloc:

  1. the part before the comma is the rows you want
  2. the part after the comma is the columns you want to select

Let’s demonstrate combining conditions and multiple column selection:

condition = (df_titanic["Age"] > 65) & (df_titanic['Sex'] == 'male')
columns = ["Name", 'Pclass']

seniors = df_titanic.loc[condition, columns]

Summary: Indexing / selection

The basics of indexing are as follows:

Operation Syntax Result
Select column df[col] Series
Select row by label df.loc[label] Series
Select row by integer location df.iloc[loc] Series
Slice rows df[5:10] DataFrame
Select rows by boolean vector df[bool_vec] DataFrame

Source: https://pandas.pydata.org/docs/user_guide/dsintro.html#indexing-selection

Vector-scalar operations

Since the Height and Weight are in pounds, and the formula assumes Metric System, we need to convert the Imperial System numbers first:

df_socr['Height(meters)'] = df_socr['Height(Inches)'] * 0.0254
df_socr['Weight(kilograms)'] = df_socr['Weight(Pounds)'] * 0.453592
df_socr.head()

Notice how we are multiplying a scalar like 0.0254 by a vector (Series). This is done element-wise (on each element), known as Broadcasting the operation.

df_socr['BMI'] = df_socr['Weight(kilograms)'] / (df_socr['Height(meters)'] ** 2)
