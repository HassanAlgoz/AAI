  5. Solve the first question
  6. Do a git add student/chap01.ipynb again, followed by a git commit -m "C1 Chapter 1: q1" (or via GUI)
  7. Repeat for next questions (steps 4 and 5)
  8. Repeat for next exercise notebooks (steps 1 to 6)

Note: this work should be submitted by the end of today 11:59 PM. So, make sure your commits are pushed before that time: git push (or via GUI).
