1. Programming as a Way of Thinking

1.1 Arithmetic operators

Arithmetic operators perform basic math on numbers. In Python, the main arithmetic operators are: - + (addition), - (subtraction) - * (multiplication), / (true division) - // (integer/floor division), % (remainder/modulus) - ** (exponentiation)

These operators allow you to perform mathematical computations in your code.

1.2 Expressions

An expression is any piece of code that produces a value when Python evaluates it. Examples include 2 + 3, len("hi"), or x > 0. Python evaluates expressions to produce a result that can be stored in a variable, printed, or used in other expressions.

A string is a sequence of characters enclosed in quotes (single '...' or double "..."). Strings support operations like concatenation (+) and repetition (*). Strings are immutable, meaning you cannot change them in place, but you can create new strings.
