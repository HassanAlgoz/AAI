# B5: Applied AI Bootcamp

## Courses

| #  | Course                                  | Description                                                                                               |
|----|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| 0  | [Introductions](/courses/00_Intros/)                | Introductions |
| 1  | [Fundamentals](/courses/01_Fundamentals/)           | Build strong Python programming foundations to apply and integrate ML/DL/AI in software systems.           |
| 2  | [Agentic Engineering](/courses/W6/)                 | Maximize the use of *Vibe Coding* via best practices to accelerate software development in your codebase.  |
| 3  | [Applied Data Science](/courses/02_Data_Science/)   | Calculate, analyze, visualize, and interpret data. Then, formulate and test hypotheses.                   |
| 4  | [Applied Machine Learning](/courses/W3/)            | Build reliable predictive modeling pipelines, debug its issues, evaluate and compare alternatives.         |
| 5  | [Agentic AI](/courses/W4/)                          | Develop, debug, evaluate, deploy, and monitor LLM-driven AI Agents to automate tasks involving unstructured data. |
| 6  | [Applied Deep Learning](/courses/W5/)               | Use DL solution frameworks / libraries to apply AI to specific problems in NLP (language) and Computer Vision.   |

**Estimated Duration**: 8 weeks.
