# Data Science Quiz

Select the single best answer for each question.

---

1. Which description most accurately characterizes data science as a discipline?
   - A) A statistical inference practice applied to cleaned numerical tables.
   - B) A blend of mathematics, computing, and subject-matter expertise.
   - C) An automated pipeline that minimizes the need for human input.
   - D) A visualization workflow that summarizes historical business metrics.

2. Within the data science Venn diagram, the "domain knowledge" component primarily enables a practitioner to:
   - A) implement durable storage systems for large unstructured datasets.
   - B) derive mathematical guarantees about model error bounds.
   - C) frame meaningful questions and interpret results in context.
   - D) optimize parallel computation across distributed clusters.

3. Which responsibility belongs most cleanly to a data engineer rather than to an analyst or an ML engineer?
   - A) Building reliable pipelines that deliver clean data to other teams.
   - B) Designing dashboards that report current key performance indicators.
   - C) Training algorithms that forecast outcomes from historical features.
   - D) Communicating findings about past performance to business leaders.

4. A teammate is engineering features and removing duplicate rows. This work belongs to which stage of the data science lifecycle?
   - A) Collection of raw inputs from external sources.
   - B) Preprocessing of acquired data for downstream use.
   - C) Analysis of patterns through visualization techniques.
   - D) Deployment of validated models into production environments.

5. Which sequence correctly orders the standard data science pipeline?
   - A) Preprocessing, Collection, Modeling, Analysis, Deployment.
   - B) Modeling, Collection, Preprocessing, Analysis, Deployment.
   - C) Collection, Preprocessing, Analysis, Modeling, Deployment.
   - D) Collection, Analysis, Preprocessing, Deployment, Modeling.

6. The defining purpose of exploratory data analysis is to:
   - A) confirm a predetermined hypothesis using formal statistical tests.
   - B) surface patterns and anomalies before any hypothesis exists.
   - C) serve trained models so they can act on live records.
   - D) document final conclusions for executive stakeholders.

7. Confirmatory data analysis is best described as:
   - A) generating new questions from previously unseen datasets.
   - B) cleaning raw inputs so they become machine readable.
   - C) applying formal tests against falsifiable hypotheses.
   - D) summarizing trends through descriptive visualizations.

8. The lesson's "wide net versus targeted arrow" analogy emphasizes that:
   - A) confirmation discards data while exploration preserves all observations.
   - B) exploration begins without a target, while confirmation begins with one.
   - C) confirmation requires larger sample sizes than exploration generally does.
   - D) exploration depends on machine learning while confirmation depends on rules.

9. Concluding from one news story that "all swordfish are dangerous" most directly shows a failure to:
   - A) collect every available record before cleaning the dataset.
   - B) test a specific claim using confirmatory analysis.
   - C) deploy a model before presenting business findings.
   - D) visualize historical data before asking questions.

10. A claim that a drug killed someone because one patient died shortly after taking it most needs:
    - A) a dashboard that reports the current number of patients.
    - B) a prediction about whether future patients will click an ad.
    - C) a formal test of a clear, falsifiable hypothesis.
    - D) a storage pipeline for larger unstructured data files.

11. Predictive analytics is best understood as the prediction of:
    - A) future business outcomes derived strictly from time-series records.
    - B) unknown events that may lie in any temporal direction.
    - C) trends extracted by descriptive statistical summaries.
    - D) deterministic processes governed by fixed physical laws.

12. Flagging a credit card transaction as fraudulent at the moment it occurs is an example of predictive analytics applied to:
    - A) a future event that has not yet been triggered.
    - B) a present event whose true label is still unknown.
    - C) a historical event already labeled in the database.
    - D) a hypothetical event constructed solely for simulation.

13. A bimodal histogram of trip distance with no mid-range rides most plausibly suggests that:
    - A) the city's customer base segments into two distinct travel patterns.
    - B) the application is silently dropping every mid-range trip from its logs.
    - C) the company's fare structure forbids medium-distance journeys outright.
    - D) the data file has been merged from two incompatible source schemas.

14. When a missing-value heatmap shows drop-off location data absent 100% for rides booked between 2 AM and 4 AM, the most informative next step is to:
    - A) conclude that night-time riders never choose a destination.
    - B) assume the city has no rides during that time window.
    - C) investigate whether a recurring operational issue affects that window.
    - D) report that drop-off locations are missing at random.

15. According to the lesson's tree diagram, data science branches into three top-level kinds of analysis labeled:
    - A) descriptive, prescriptive, and diagnostic.
    - B) quantitative, qualitative, and mixed-methods.
    - C) exploratory, confirmatory, and predictive.
    - D) supervised, unsupervised, and reinforcement.
